# Young Kids Lesson Plans (Ages 5-11)

## Program Overview

**Duration:** 8 weeks (with flexibility for shorter attention spans)  
**Lesson Length:** 30-40 minutes (not full 60 min - kids' attention span)  
**Format:** Game-based, visual, interactive, movement-based  
**Proficiency Level:** A0-A1 (Absolute Beginner to Basic)

---

## Key Teaching Principles for Young Kids

✅ **Short, frequent lessons** (30-40 min)  
✅ **Games & play** (primary teaching method)  
✅ **Visual learning** (pictures, colors, movement)  
✅ **Songs & rhymes** (memory aid)  
✅ **Physical activity** (Total Physical Response - TPR)  
✅ **Praise constantly** (build confidence)  
✅ **No grammar explanation** (learn through immersion)  
✅ **Colorful materials** (keep it fun!)  
✅ **Repetition** (kids learn through repetition)  
✅ **Parent involvement** (at least for younger kids)

## Week 1: Colors, Shapes & Greetings

### Lesson 1.1: Hello & Colors (30 minutes)

**Objectives:**
- Recognize greetings (Hello, Hi, Good morning)
- Name basic colors (red, blue, yellow, green)
- Respond to simple questions

#### Activity 1: Greeting Dance (5 min)
```
Teacher: "Hello! Good morning!"
Kids wave, dance, jump
Teacher demonstrates movement
Kids copy
Repeat 5 times with enthusiasm
```

**Why:** Movement helps memory, fun = engagement

#### Activity 2: Color Hunt (10 min)
```
Teacher shows color card (RED)
Teacher: "What color is this?" (Point dramatically)
Kids: "RED!"
Kids run around room finding RED objects
Touch red chair, red pencil, red shirt, etc.
Bring items back to circle
Repeat with: BLUE, YELLOW, GREEN
```

**Materials Needed:**
- Large color cards
- Items in each color

#### Activity 3: Color Song (8 min)
```
Sing to tune of "Twinkle Twinkle Little Star":

"Red, red, red is the color,
Beautiful red, so bright!
Blue, blue, blue is the color,
Beautiful blue, so bright!
Yellow, yellow, yellow,
Green, green, green!
Prettiest colors ever seen!"

Clap hands on each color word
Kids join in
Repeat 2-3 times
```

#### Activity 4: Color Game - "Simon Says" (5 min)
```
Simon says, "Touch something BLUE!"
Kids run and touch blue thing
Simon says, "Jump on something YELLOW!"
Kids jump
Teacher does wrong action (no Simon says)
Kids should NOT copy
Repeat with different colors
Winner = most accurate
```

#### Closing Circle (2 min)
```
Sit together
Teacher: "What colors did we learn today?"
Point to objects
Kids say color names
Give stickers/praise
"Great job today! See you next time!"
```

**Homework for Parents:**
- Point to 5 colored objects at home
- Ask child the color
- Practice song together

### Lesson 1.2: Shapes & Body Parts (30 minutes)

**Objectives:**
- Name basic shapes (circle, square, triangle)
- Name body parts (head, arms, legs, hands)

Teacher: "What shape is this?" "SQUARE!"

Teacher: "Draw a BIG CIRCLE!"
Kids draw
Teacher: "Draw a small TRIANGLE!"
Kids draw
Kids stand and show their drawings
Teacher praises
Repeat with square, more complex shapes

```

#### Activity 3: Body Parts Song (8 min)

Song to "If You're Happy and You Know It":

"Point to your HEAD, head, head!
Point to your ARMS, arms, arms!
Point to your LEGS, legs, legs!
Point to your HANDS, hands, hands!"

Kids do actions while singing
Teacher models each movement
Faster each time = fun challenge
```

#### Activity 4: Simon Says - Body Parts (5 min)
```
Simon says, "Touch your HEAD!"

"Touch your NOSE!" (no Simon says - kids shouldn't copy)
Kids who copy = silly, everyone laughs
No winner/loser, all for fun
```

#### Closing (1 min)
```
High-fives for everyone
"Excellent work!"
```

### Lesson 1.3: Numbers 1-5 & Counting (30 minutes)

**Objectives:**
- Count from 1-5
- Recognize written numbers
- Do number-related physical activities

#### Activity 1: Number Song (8 min)
```

"One, one, can you count?
Two, two, jump and shout!
Three, three, clap your hands!
Four, four, stand like trees!
Five, five, touch the sky!
One, two, three, four, five!
Counting high, counting high!"

Kids do actions for each number
Dance/jump/spin

Repeat slower, then faster
```

#### Activity 2: Number Hunt (10 min)
```
Hide number cards (1, 2, 3, 4, 5) around room
Teacher: "Find number ONE!"
Kids search, bring card back
Teacher: "How many fingers for number ONE?"
Kids hold up 1 finger
Repeat for all 5 numbers
```

#### Activity 3: Counting Game (8 min)
```
Teacher shows fingers: "How many?"
Kids count: "ONE! TWO! THREE!"
Teacher hops: "Count while I hop!"
Kids count: "ONE! TWO! THREE! FOUR! FIVE!"
Kids take turns being the counter
Lots of movement + fun
```

#### Activity 4: Number Coloring (3 min)
```
Give worksheet with numbers
Kids count dots and color numbers
Easy, fun, celebrates writing
```

#### Closing (1 min)
```
Review: Count together 1-5
"You're such amazing counters!"
Stickers
```

## Week 2: Animals & Sounds

### Lesson 2.1: Farm Animals & Sounds (30 minutes)

**Objectives:**
- Name farm animals
- Make animal sounds
- Follow commands (animal-themed)
- Gain confidence speaking English

#### Activity 1: Animal Introduction (8 min)
```
Teacher shows picture: COW
"What animal is this?"
Kids: "COW!"
Teacher: "What sound does a cow make?"
Kids: "MOOOOO!"
Repeat with: SHEEP, PIG, DUCK, DOG

Each introduction:
- Show picture
- Say animal name
- Make sound together
- Do animal movement (hop for rabbit, etc.)
```

#### Activity 2: Animal Movement Game (10 min)
```
Teacher: "Can you walk like a DUCK?"
Kids waddle like ducks, quacking
Teacher: "Now hop like a RABBIT!"
Kids hop
Teacher: "Gallop like a HORSE!"
Kids run around making horse sounds
Teachers call increasingly silly combinations:
"Walk like a duck while mooing like a cow!"
"Hop like a rabbit while barking like a dog!"
Kids love the silliness
```

#### Activity 3: Animal Sound Guessing Game (8 min)
```
Teacher makes animal sound
Kids guess: "Is it a COW?" "Is it a SHEEP?"
First correct guess wins
Kids take turns making sounds
Others guess
Lots of giggling and learning
```

#### Activity 4: Farm Drawing (3 min)
```
Give paper + crayons
"Draw your favorite farm animal!"
Kids draw
Teacher goes around: "What animal is this?"
Kids say animal name
Hang up and celebrate
```

#### Closing (1 min)
```
Review animal names and sounds
"Fantastic job making all those sounds!"
Stickers
```

### Lesson 2.2: Wild Animals (30 minutes)

**Objectives:**
- Learn wild animals (lion, tiger, elephant, monkey, snake)
- Describe what animals do
- More complex sounds/movements

#### Activity 1: Wild Animal Introduction (8 min)
```
Show picture: LION
"What animal?"
Kids: "LION!"
"What does a lion do?"
Teacher: "ROAR!" (big dramatic roar)
Kids: "ROAR!" (even louder)
Emphasize the SOUND

Repeat with:
TIGER - ROAR but striped
ELEPHANT - Makes trumpet sound (trunks up)
MONKEY - Ooh ooh ooh + jump around
SNAKE - SSSSSSSSS while wiggling on ground
```

#### Activity 2: Wild Animal Actions (10 min)
```
Teacher: "Crawl like a SNAKE!"

Teacher: "Swing like a MONKEY!"

Teacher: "Stomp like an ELEPHANT!"
Kids heavy steps, trumpet sound
Teacher: "Roar like a TIGER!"
Kids pounce, roar
Teacher: "Slink like a LION!"
Kids creep around quietly, then ROAR
```

#### Activity 3: Animal Charades (10 min)
```
One kid acts out animal silently
Others guess: "Is it a tiger?"
First correct guess wins
Each kid gets a turn
Very engaging for this age
```

#### Activity 4: Wild Animal Drawing (1 min)
```
Quick draw favorite wild animal
Celebrate
```

#### Closing (1 min)
```
"Best wild animal explorers!"
Stickers/certificates
```

## Week 4: Food & Eating

### Lesson 4.1: Fruits (30 minutes)

**Objectives:**
- Name fruits (apple, banana, orange, grape, strawberry, watermelon)
- Describe colors
- "I like" sentences

#### Activity 1: Fruit Introduction (8 min)
```
Show real fruit or picture: APPLE
"What is this?"
Kids: "APPLE!"
"What color?"
Kids: "RED!"
Repeat with:
BANANA (yellow), ORANGE (orange), GRAPE (purple), 
STRAWBERRY (red), WATERMELON (green)

Each fruit:
- Show picture/real
- Name + color
- Make sound of eating (crunch, slurp, etc.)
- Kids repeat
```

#### Activity 2: Fruit Song (8 min)
```
Sing to "Twinkle Twinkle Little Star":

"Apple, apple, red and round,
Sweetest fruit that can be found!
Banana yellow, long and nice,
Orange, grape, and strawberry rice!
Watermelon big and green,
Best fruits you have ever seen!"

Actions for each fruit:
APPLE - hold in hand, bite
BANANA - peel and eat
ORANGE - squeeze
GRAPE - pop in mouth
STRAWBERRY - pick small
WATERMELON - slice with hands
```

#### Activity 3: Fruit Taste Game (10 min)
```
Teacher brings small fruit pieces
"This is APPLE. Do you like apple?"
Kids: "Yes! I like apple!"
"I don't like apple." (some kids)
Teacher: "OK! Some people like apple, some don't."
Kids taste and say:
"Mmmmm! I like APPLE!"
Or: "Yuck! I don't like apple."
Build natural language patterns
```

#### Activity 4: Fruit Drawing (3 min)
```
"Draw your favorite fruit!"
Colors
Celebrate
```

#### Closing (1 min)
```
"Fruit experts!"
Stickers
```

## Special Features

### Behavior Management for Young Kids

**Positive Reinforcement:**
- Stickers after each lesson (and at end of class)
- Certificates for each week and full program
- Praise specific behaviors: "Great listening!" not just "Good job!"
- High-fives, fist bumps, special handshakes

**Engagement Strategies:**
- Change activities every 5-8 minutes (kids' attention span)
- Movement between every activity
- Include games, songs, drawing, acting
- Let kids choose sometimes
- Celebrate mistakes: "Nice try!"

**Classroom Management:**
- Clear simple rules (Listen, Respect, Have Fun)
- Consistent start/ending ritual
- Predictable lesson structure
- Positive choices, not punishments

### Parent Involvement

**Send Home:**
- Weekly homework sheet with simple activities
- Song lyrics (for practice at home)
- Vocabulary list
- Suggestion for "English time" (10-15 min daily)

**Parent Tips:**
- Don't force learning - keep it fun
- Praise all attempts
- Use visuals (pictures, gestures)
- Repeat songs and games
- Connect to real life: "Look, a RED car!"

### Assessment for Young Kids

**NOT Tests - Instead:**
- Observe what kids understand
- Can they respond to commands?
- Do they say words?
- Are they engaged?
- Do they try to communicate?

**Simple Checklists (For Teacher):**
- Week 1: Knows colors ✓ / Says animal names ✓
- Week 4: Likes/dislikes foods ✓ / Can count 1-5 ✓
- Week 8: Tells simple story ✓ / Confident speaking ✓

**Celebration:**
- Every child "graduates"
- Every child receives certificate
- Every child is hero of their own learning journey
