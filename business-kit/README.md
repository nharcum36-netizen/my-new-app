# Confidence English Academy - Complete Launch Kit

This folder contains a full business launch system for an English tutoring service.

## What you have

- `business-kit/website-copy.md` - ready-to-paste website copy
- `business-kit/pricing-and-offers.md` - final pricing model and package structure
- `business-kit/brand-and-positioning.md` - mission, niche, and value proposition
- `business-kit/curriculum/8-week-roadmap.md` - structured lesson path by level
- `business-kit/operations/sop.md` - scheduling, follow-ups, and student tracking workflow
- `business-kit/operations/legal-admin-checklist.md` - launch compliance checklist
- `business-kit/marketing/14-day-content-calendar.csv` - post calendar for TikTok/IG/YouTube Shorts
- `business-kit/marketing/social-posts-and-scripts.md` - first scripts and captions
- `business-kit/marketing/email-sequence.md` - 3-email lead nurture sequence

## Fast launch checklist

1. Publish website copy and pricing.
2. Connect booking (Calendly) and Zoom.
3. Set payment links (Stripe + PayPal backup).
4. Post daily using the 14-day calendar.
5. Collect testimonials from first 5 students.
6. Run weekly review of leads, trials, conversions.
